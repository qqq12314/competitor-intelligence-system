# 基于 DeepSeek 与 LangChain 的竞品动态追踪与智能对标分析系统

## 项目简介
本项目面向企业竞品情报分析场景，围绕价格战、新品发布、负面舆情三大维度，构建竞品动态追踪、智能对标分析和竞争态势简报生成系统。

## 技术栈
- 后端：FastAPI
- 前端：Vue 3 / React
- 大模型：DeepSeek API
- LLM 框架：LangChain
- 向量库：Chroma
- 数据库：SQLite / MySQL
- 可视化：ECharts
- 部署：Docker

## 核心功能
- 竞品主体管理
- 舆情数据导入
- RAG 检索增强分析
- 三维度竞品动态识别
- 机会点与威胁点分析
- 竞争态势简报生成
- 前端可视化看板

# 茶饮咖啡竞品分析系统

本项目面向中国现制茶饮、现制咖啡与茶咖融合赛道，围绕品牌定位、产品结构、价格策略、渠道布局、营销传播、用户反馈和经营信号七大维度，构建一个轻量化竞品数据分析与智能报告生成原型。

项目目标是在两周内完成一个可演示版本：能够管理茶饮咖啡品牌样本，导入竞品数据，计算指标评分，展示可视化看板，并通过 DeepSeek 生成品牌画像、竞品对比结论和 Markdown 分析报告。

## 核心功能

- 竞品品牌库：维护不少于 10 个茶饮咖啡品牌的基础信息。
- 数据导入：支持 CSV 样例数据导入与字段清洗。
- 指标评分：按七大维度输出品牌评分、综合分和风险等级。
- 可视化看板：展示价格带、品牌定位矩阵、雷达图和评分排行。
- 智能摘要：调用 DeepSeek 生成优势、压力、机会点、风险点和策略建议。
- 报告生成：生成可预览的 Markdown 竞品分析报告。

## 技术栈

- 后端：FastAPI
- 数据库：SQLite，后期可切换 MySQL
- 数据处理：pandas
- 大模型：DeepSeek API
- LLM 工程：LangChain PromptTemplate / Structured Output
- 前端：Vue 3 + Vite + Element Plus + ECharts
- 测试：pytest + httpx

## 项目结构


第一周完成后端骨架、品牌库、数据导入、指标评分、前端基础页面和前后端联调。第二周完成图表看板、DeepSeek 智能摘要、报告生成、测试优化、项目文档和最终演示材料。

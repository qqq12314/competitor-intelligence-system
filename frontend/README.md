# 前端说明

前端计划使用 Vue 3 + Vite + Element Plus + ECharts，实现茶饮咖啡竞品分析看板。

## 页面规划

- 首页看板：展示品牌总数、平均价格带、风险品牌、营销事件数量和评分排行。
- 竞品品牌库：展示品牌列表、品牌详情和编辑入口。
- 数据导入页：上传 CSV 样例数据并展示导入结果。
- 分析看板页：展示价格带对比、七维雷达图、品牌定位矩阵和营销事件时间线。
- 智能报告页：调用后端生成 Markdown 分析报告并预览。

## 后续启动方式

Day 6 搭建正式 Vue 项目后，在 `frontend/` 目录执行：

```powershell
npm install
npm run dev
```

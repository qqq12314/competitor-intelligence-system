# TS 爬虫数据接入说明

本目录先放置基于 TS 已提供爬虫结果整理出的聚合样例数据，用于后端第一阶段接口联调和前端展示。

现有爬虫数据的定位是“茶饮咖啡行业公开经营环境数据”，适合用于品牌覆盖、城市门店分布、区域竞争密度、新闻舆情和品牌风险样例分析。它不能直接替代真实信贷审批数据。

后续若 TS 提供完整 CSV，可继续补充：

- `store_clean.csv`：门店原始清洗数据，字段包括品牌、门店名称、省份、城市、区县、地址、经纬度、电话、评分、人均消费、营业时间、POI 类型。
- `brand.csv`：品牌基础数据。
- `news.csv`：品牌新闻和舆情数据。
- `risk_data.csv`：品牌风险样例。
- `merchant_credit_sample.csv`：商户月流水、订单量、复购率、租金、人工、原料成本、负债、贷款金额等信贷核心字段。
- `contract_sample.csv`：商户租赁、加盟、采购等合同文本样本。

当前后端会在启动时自动导入本目录的聚合样例 CSV 到 SQLite，用于 `/api/spider/*` 和 `/api/ai/*` 接口演示。

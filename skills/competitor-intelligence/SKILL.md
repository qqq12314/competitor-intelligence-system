# Tea Coffee Competitor Intelligence

本 Skill 用于把茶饮咖啡行业的品牌资料、产品价格、营销事件、用户评价和经营信号整理为结构化竞品情报，并输出可用于看板展示和报告生成的分析结果。

## 输入

- 品牌基础信息：品牌名称、类别、价格带、目标人群、品牌定位、门店规模。
- 产品信息：主力产品、新品频率、季节限定、健康标签。
- 价格信息：基础价格、优惠价格、券包、会员权益。
- 渠道信息：门店规模、城市层级、商圈类型、外卖平台覆盖。
- 营销信息：联名活动、社媒声量、私域运营、达人内容。
- 用户反馈：评分、评价关键词、差评原因、复购倾向。
- 经营信号：开闭店、财报披露、融资上市、供应链建设。

## 分析流程

1. 识别品牌所属类别：大众茶饮、中高端茶饮、大众咖啡、品质/场景咖啡。
2. 按七大维度抽取核心指标，并统一字段口径。
3. 计算品牌维度评分、综合评分和风险等级。
4. 对比目标品牌与竞品在价格、产品、渠道、营销和用户反馈上的差异。
5. 生成品牌画像、主要优势、主要压力、机会点、风险点和策略建议。
6. 保留数据来源、采集日期和人工修正说明，保证结论可追溯。

## 输出格式

```json
{
  "brand": "瑞幸咖啡",
  "summary": "瑞幸咖啡具备较强的数字化运营、爆品打造和门店扩张能力。",
  "strengths": ["数字化下单成熟", "上新节奏快", "私域运营强"],
  "pressures": ["补贴成本压力", "同店增长质量需要观察"],
  "opportunities": ["继续强化轻咖和茶咖融合产品", "提高会员复购"],
  "risks": ["低价竞争持续", "外卖平台费用波动"],
  "suggestions": ["跟踪新品复购表现", "对比库迪和星巴克的价格与场景策略"]
}
```

## 适用场景

- 生成茶饮咖啡品牌画像。
- 对比多个品牌的价格带、产品结构和营销策略。
- 从用户评价中提取正负面关键词。
- 生成竞品分析报告和答辩摘要。

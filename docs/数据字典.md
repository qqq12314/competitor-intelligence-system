# 数据字典

本项目围绕茶饮咖啡竞品分析建立统一字段口径。Day 1 先定义品牌基础信息表，后续再扩展产品、评价和营销事件表。

## 品牌基础信息表

| 字段 | 含义 | 示例 |
|---|---|---|
| `brand_name` | 品牌名称 | 瑞幸咖啡 |
| `category` | 品牌类别 | 大众咖啡 |
| `price_band` | 主流价格带 | 9-20元 |
| `target_users` | 目标用户 | 通勤白领与高频咖啡用户 |
| `positioning` | 品牌定位 | 数字化运营与爆品驱动咖啡品牌 |
| `store_scale` | 门店规模描述 | 超3万家 |
| `main_products` | 主力产品 | 拿铁/生椰拿铁/季节咖啡 |
| `new_product_frequency` | 新品频率 | 高 |
| `marketing_style` | 营销特点 | App私域与补贴活动 |
| `user_rating` | 样本评分 | 4.6 |
| `strengths` | 主要优势 | 数字化强/上新快/门店扩张快 |
| `pressures` | 主要压力 | 补贴压力/同店增长质量 |
| `benchmark_notes` | 对标启示 | 咖啡品牌可学习产品运营与私域转化 |

## 七维指标

| 维度 | 说明 | 代码端输出 |
|---|---|---|
| 品牌定位 | 价格带、目标人群、主张关键词、门店形象 | `brand_position_score` |
| 产品结构 | SKU、新品频率、主力品类、健康标签 | `product_score` |
| 价格策略 | 基础杯单价、优惠价、券包、会员权益 | `price_score` |
| 渠道布局 | 门店规模、城市层级、商圈类型、外卖覆盖 | `channel_score` |
| 营销传播 | 联名、达人内容、社媒声量、私域运营 | `marketing_score` |
| 用户反馈 | 评分、评价关键词、差评原因、复购倾向 | `feedback_score` |
| 经营信号 | 开闭店、财报、融资上市、供应链建设 | `business_signal_score` |
